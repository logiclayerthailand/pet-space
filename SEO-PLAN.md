# แผน SEO — Pet Space Grooming & Hotel (บางพลี สมุทรปราการ)

## 1. เป้าหมาย
ให้ร้านติดอันดับ Google เมื่อคนย่าน **บางพลี / สมุทรปราการ / บางนา / สุวรรณภูมิ** ค้นหา
บริการอาบน้ำ-ตัดขน-สปา-รับฝากสัตว์เลี้ยง และดึงให้ทักผ่าน LINE @pet-space / โทร

## 2. Keyword Map (target ต่อหน้า)

| หน้า | Primary keyword | Secondary / Long-tail |
|------|-----------------|------------------------|
| Home | ร้านอาบน้ำตัดขนสุนัข บางพลี | อาบน้ำตัดขนสุนัข สมุทรปราการ, ตัดขนสุนัข ใกล้ฉัน, grooming บางนา |
| อาบน้ำ-ตัดขนสุนัข | อาบน้ำตัดขนสุนัข ราคา | ตัดขนพุดเดิ้ล/ชิสุ/ปอม บางพลี, ตัดขนหมาพันธุ์ใหญ่ |
| อาบน้ำแมว | อาบน้ำแมว บางพลี | อาบน้ำแมวขนยาว, ตัดขนแมว สมุทรปราการ |
| สปา ARTERO | สปาสุนัข สมุทรปราการ | detox สุนัข, keratin ขนสุนัข, สปาแมว ARTERO |
| โรงแรม/ฝากเลี้ยง | รับฝากสุนัข บางพลี | pet hotel บางพลี, ที่รับฝากหมาแมว สมุทรปราการ, ฝากเลี้ยงหมา ใกล้สุวรรณภูมิ |
| ติดต่อ | Pet Space บางพลี | ร้านสัตว์เลี้ยง บางพลีใหญ่ |

**Search intent:** ส่วนใหญ่เป็น local + transactional → ต้องมี ราคา, ที่อยู่, ปุ่มติดต่อ ชัดเจนทุกหน้า

## 3. On-Page checklist (ทำครบทุกหน้า)
- [ ] `<title>` มี keyword + แบรนด์ + location (≤ 60 ตัวอักษร)
- [ ] `meta description` ชวนคลิก + มีเบอร์/พื้นที่ (≤ 155 ตัวอักษร)
- [ ] H1 เดียวต่อหน้า มี primary keyword
- [ ] H2/H3 ครอบ secondary keyword (ตารางราคา, บริการ, FAQ)
- [ ] รูปทุกใบมี `alt` เชิงบรรยาย + keyword (เช่น "ตัดขนพุดเดิ้ลทรง Teddy บางพลี")
- [ ] Internal link ระหว่างหน้า (บริการ ↔ ราคา ↔ ติดต่อ)
- [ ] ปุ่ม LINE/โทร แบบ click-to-call `tel:` และลิงก์ LINE
- [ ] URL อ่านง่าย เช่น /dog-grooming, /cat-grooming, /spa, /pet-hotel

## 4. Structured Data (JSON-LD) — ฝังใน index.html แล้ว
- **LocalBusiness / PetStore**: NAP + geo + openingHours (Sa–Th 10–19, ปิดศุกร์) + telephone + priceRange
- **OfferCatalog / Service**: รายการบริการ + ราคา
- **FAQPage**: คำถามยอดฮิต (เปิดกี่โมง, รับพันธุ์ใหญ่ไหม, ราคาเท่าไหร่, ฝากเลี้ยงได้ไหม)
- **BreadcrumbList** (เมื่อแตกหลายหน้า)

## 5. Local SEO (นอกเว็บ — ทำคู่กัน สำคัญพอๆ กับเว็บ)
1. **Google Business Profile** — ตั้งค่าให้ครบ: หมวด "Pet groomer" + "Pet boarding service", ที่อยู่ตรงกับเว็บเป๊ะ, เวลาเปิด, รูป 10+ ใบ, ตอบรีวิว
2. NAP ให้ **ตรงกันทุกที่** (เว็บ, Facebook, Google, LINE) — สะกด "111/13 บางพลีใหญ่ บางพลี สมุทรปราการ 10540" เหมือนกันหมด
3. เก็บรีวิว Google Maps ต่อเนื่อง (มีโปรลด 50 อยู่แล้ว — ใช้ประโยชน์)
4. ลงไดเรกทอรีสัตว์เลี้ยง/ท้องถิ่นไทย + กลุ่ม Facebook คนเลี้ยงหมาแมวบางพลี/บางนา

- "ตัดขนพุดเดิ้ลทรงไหนดี? รวมทรงยอดฮิต Toy/Miniature/Standard"

- "ราคาตัดขนสุนัขตามขนาด XS–L อัปเดตล่าสุด"

## 7. Technical
- Mobile-first (ลูกค้าเสิร์ชจากมือถือเกือบ 100%), โหลดเร็ว, รูป lazy-load + webp
- HTTPS, sitemap.xml, robots.txt
- Open Graph + Twitter Card (ตอนแชร์ LINE/FB ให้ขึ้น preview สวย)
